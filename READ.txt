start this server : 
    uvicorn api:app --reload (This helps to fetch the local files in Next Js)

To start the app:
    cd neuro_metric> npm run dev (Dashboard Metric viewer)
              

Parameter	    Type	    Required	    Description
session_dir	    Path        Yes             Root folder of the session
age	            int     	Yes	            Patient age
sex	            str	        Yes	            "M" or "F"
interval_days	float	    Yes	            Days between T0 and T1 scan


You provide only:
    session_dir/input/T0.nii.gz
    session_dir/input/T1.nii.gz
    age
    sex
    interval_days
